## WBEMextras — Archived Project

> **Note:** WBEMextras is a retired project. No new development is planned. Critical security fixes may still be issued on request, but general updates are no longer provided.

[wbemextras](https://wbemextras.github.io/) contains Korn shell scripts to assist with the installation, upgrade, and configuration of HP SIM-related software on HP-UX systems. A SIM Health Check script inspects the status of the installed software, performs basic system checks, and can send a test event to HP Support Center. An HTML report is generated with pre-requisites and usage information.

Stable releases are available from the [wbemextras download page](https://wbemextras.github.io/download/).
